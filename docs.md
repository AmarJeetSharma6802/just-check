## 1 . CI (Continuous Integration)

```
CI ka matlab:
push hua → automatic test / build run
```
- Example:

- npm install

- npm run build

- test run

- Sirf check karta hai code sahi hai ya nahi.

- Deploy nahi karta.

##  2. CD (Continuous Deployment)

```
CD ka matlab:
build ke baad automatic deploy
```

```
Example:
push → build → deploy server
``
Matlab production update automatically.
